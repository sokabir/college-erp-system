# Portable Package Guide - Run Without Installation

## Option 2: Create Portable Package for Offline Demo

This creates a package that can run on any Windows computer without installation.

---

## What You'll Create

A folder that contains:
- ✅ Portable Node.js
- ✅ Portable MySQL
- ✅ Your application
- ✅ One-click startup script
- ✅ Pre-configured database

**Result:** Double-click `START.bat` and everything runs!

---

## Step 1: Download Portable Tools

### 1.1 Portable Node.js
1. Download: https://nodejs.org/dist/v18.17.0/node-v18.17.0-win-x64.zip
2. Extract to: `CollegeERP_Portable/nodejs/`

### 1.2 Portable MySQL
1. Download: https://dev.mysql.com/get/Downloads/MySQL-8.0/mysql-8.0.34-winx64.zip
2. Extract to: `CollegeERP_Portable/mysql/`
